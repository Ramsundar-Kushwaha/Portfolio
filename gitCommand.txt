1. clone - cloning a repository on our local machine
  git clone <someLinks>

2. status - display the state of the code
  git status

3. add - to add file at staged, so that it can be commited later
  git add fileName  (for individual file)
  git add .         (for all file at onece)

4. commit - to confirm the changes/midification
  git commit -m "some message" (this is compulsury to put message)

5. push command - upload local repo content to remote repo
  git push origin main
